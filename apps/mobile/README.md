# Novatra Mobile (`apps/mobile`)

React Native (Expo) mobile app for Novatra.

## What this app is

Mobile-first experience for users in emerging markets and diaspora users who prefer mobile.

## What we’re building (phased)

Phase 1 (MVP parity basics):

- login/onboarding
- view transfer status
- beneficiary details + history

Phase 2:

- full sender flow (quote → confirm → pay)
- recipient wallet balance (if custodial model)

## How it connects

- Talks to the API in `apps/api`.
- Will share types via `@novatra/shared`.
