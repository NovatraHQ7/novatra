# Novatra Monorepo

This repo hosts Novatra’s:

- `apps/web` (Next.js)
- `apps/api` (NestJS)
- `apps/mobile` (React Native + Expo)
- `packages/shared` (shared types/DTOs/utils)
- `packages/stellar` (Stellar settlement foundation)

## Requirements

- Node.js 20+ recommended
- pnpm 9+

## Getting started

```bash
pnpm install
pnpm dev
```
