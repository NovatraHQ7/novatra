# Shared Package (`packages/shared`)

Shared types, DTOs, and validation used across:

- `apps/web`
- `apps/api`
- `apps/mobile`
- settlement layer packages

## What we’re building here

Over time this package will contain:

- money primitives (amount handling, currency codes, rounding rules)
- request/response DTOs for quote/transfer APIs
- validation schemas (to keep web/mobile/api aligned)
