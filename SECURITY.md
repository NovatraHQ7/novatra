# Security Policy

## Supported branches

Security fixes are currently targeted to the default branch.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

Instead, report privately to: security@novatrahq.com

Include:

- Affected component (web, api, mobile, shared, stellar)
- Reproduction steps or proof of concept
- Impact assessment
- Suggested mitigation (if any)

You can expect an initial response within 3 business days.

## Disclosure process

- We acknowledge the report.
- We triage and assess impact.
- We develop and validate a fix.
- We coordinate disclosure timing with the reporter when possible.

## Safe harbor

We will not take legal action against good-faith security research conducted responsibly and reported through the process above.
